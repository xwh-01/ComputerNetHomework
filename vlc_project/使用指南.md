# OptTrans 使用指南

## 项目简介

OptTrans 是一个将二进制文件编码为图像的工具，支持通过图像传输数据，并能从手机翻拍的图像中正确识别、矫正和解码回原始二进制文件。

## 环境要求

- Python 3.12 
- 依赖库：Pillow, reedsolo

## 安装步骤

1. 确保已安装 Python 3.12(注意3.13不适配)
2. 使用 Python 3.12 安装项目依赖：

**Windows 使用 Python Launcher：**
```bash
py -3.12 -m pip install -r requirements.txt
```

**或者直接使用 Python 3.12：**
```bash
python3.12 -m pip install -r requirements.txt
```

**注意：** 确保使用正确的 Python 3.12 解释器，避免使用其他 Python 版本。

## 核心功能

### 1. 编码功能

将二进制文件编码为图像或视频，包含：
- **定位区**：3个7×7模块的L型定位图形，用于角度、缩放校正与透视变换
- **时序区**：辅助模块对齐
- **数据区**：蛇形路径填充，存储经RS纠错编码和掩码优化的二进制数据

### 2. 解码功能

从图像或视频中提取数据，支持：
- 自动识别定位图案
- 透视矫正
- RS纠错解码
- 还原原始二进制数据

## 使用方法

### 命令行工具使用

#### 编码文件为图像

```bash
py -3.12 opttrans.py encode <输入文件> <输出图像> [选项]
```

**选项：**
- `--module-size N`：每个模块的像素大小（默认：10）
- `--ecc N`：RS纠错码的符号数量（默认：5）

**示例：**
```bash
py -3.12 opttrans.py encode examples/input.bin examples/encoded.png
```

#### 解码图像为文件

```bash
py -3.12 opttrans.py decode <输入图像> <输出文件> [选项]
```

**选项：**
- `--module-size N`：每个模块的像素大小（默认：10）
- `--ecc N`：RS纠错码的符号数量（默认：5）

**示例：**
```bash
py -3.12 opttrans.py decode examples/encoded.png examples/output.bin
```

#### 编码文件为视频

```bash
py -3.12 opttrans.py encode-video <输入文件> <输出视频> [选项]
```

**选项：**
- `--fps N`：视频帧率（默认：1）

**示例：**
```bash
py -3.12 opttrans.py encode-video examples/input.bin examples/encoded.mp4
```

#### 解码视频为文件

```bash
py -3.12 opttrans.py decode-video <输入视频> <输出文件>
```

**示例：**
```bash
py -3.12 opttrans.py decode-video examples/encoded.mp4 examples/output.bin
```

**重要提示：** 请始终使用 `py -3.12` 来运行项目，确保使用正确的 Python 版本！

### 作为库使用

#### 编码示例

```python
from src.encoder_pillow import OptTransEncoderPillow

encoder = OptTransEncoderPillow(module_size=10, rs_ecc_symbols=5)

# 编码数据
encoder.encode_data(b"Hello OptTrans!", "output.png")

# 编码文件
encoder.encode_file("input.bin", "output.png")
```

#### 解码示例

```python
from src.decoder_pillow import OptTransDecoderPillow

decoder = OptTransDecoderPillow(module_size=10, rs_ecc_symbols=5)

# 解码图像
data = decoder.decode_image("input.png")
print(data)

# 解码到文件
decoder.decode_to_file("input.png", "output.bin")
```

```python
from src.encoder_pillow import OptTransEncoderPillow

encoder = OptTransEncoderPillow()

encoder.encode_file_to_video("input.bin", "output.mp4", fps=1)
```

## 参数说明

### module_size（模块大小）

- 控制图像中每个模块的像素尺寸
- 建议值：8-20
- 较大的值：更容易拍摄识别，但图像尺寸更大
- 较小的值：图像更紧凑，但拍摄要求更高

### rs_ecc_symbols（RS纠错符号数）

- Reed-Solomon纠错码的冗余符号数量
- 建议值：5-20
- 较大的值：纠错能力更强，但数据容量减少
- 较小的值：数据容量更大，但抗干扰能力降低

## 技术特点

1. **定位图案**：采用类QR码的3个定位角设计，支持透视变换矫正
2. **Reed-Solomon纠错**：提供强大的错误纠正能力，能容忍一定程度的图像损坏或拍摄模糊
3. **掩码优化**：改善图像的视觉特性，减少长串相同颜色导致的问题
4. **蛇形填充**：优化数据布局，减少连续错误的影响

## 注意事项

1. 拍摄图像时请确保光线充足，图像清晰
2. 尽量保持图像平面与手机摄像头平行
3. 图像中应包含完整的编码图案，四周留有适当空白
4. 建议使用较高的module_size值以便于拍摄识别

## 示例

### 二进制文件 ↔ 图片转换

**核心功能：将任意二进制文件编码为图片，再从图片解码回二进制文件**

```bash
# 1. 编码二进制文件为图片
py -3.12 opttrans.py encode examples/input.bin examples/encoded.png

# 2. 从图片解码回二进制文件
py -3.12 opttrans.py decode examples/encoded.png examples/decoded.bin

# 3. 验证结果（两个文件内容完全一致）
```

### 完整文本文件示例

```bash
# 1. 创建测试文件
echo "Hello World!" > examples/test.bin

# 2. 编码为图像
py -3.12 opttrans.py encode examples/test.bin examples/test_encoded.png

# 3. （可选）用手机拍摄图像，或直接使用生成的图像

# 4. 解码回文件
py -3.12 opttrans.py decode examples/test_encoded.png examples/test_decoded.bin

# 5. 验证结果
Get-Content examples/test_decoded.bin
```

## 故障排除

### 解码失败
- 检查图像是否清晰
- 尝试增大 module_size 重新编码
- 增加 rs_ecc_symbols 提高纠错能力
- 确保图像完整包含编码图案

### 依赖安装问题
- 确保使用 Python 3.12
- 尝试升级 pip：`python -m pip install --upgrade pip`
- 使用国内镜像源：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`
